Supabase Credentials Test



How to get correct Supabase credentials:

  1. Go to https://supabase.com/dashboard
  2. Sign in to your account
  3. Select your project (or create a new one)
  4. Go to Settings → API
  5. Copy the Project URL and anon/public key
  6. Test them using the form above
  7. Update your .env file with the correct values

Note: If you don't have a Supabase project yet, you'll need to create one first.
